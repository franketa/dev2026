# Dev2026 - Client Landing Pages

Monorepo for managing client landing pages deployed to Coolify.

## Structure

```
clients/          # Individual client landing pages
templates/        # Reusable templates and components
```

## Adding a New Client

1. Create a new folder in `clients/` with the client name
2. Add your HTML, CSS, JS, and assets
3. Deploy to Coolify using the specific client subdirectory

## Deployment

Each client folder can be deployed independently to Coolify by pointing to the specific subdirectory path.
